# Gestion-Événements

# Technologies utilisées : Node.js, Express.js, MongoDB.

Projet Node.js + Express pour gérer des événements, participants, conférenciers, salles et billetterie.

Le projet Gestion-Événements est une application web permettant de gérer l’organisation complète d’événements, depuis la planification jusqu’à la billetterie. Il fournit une interface pour les organisateurs, les participants et les conférenciers, facilitant la gestion des événements, des salles, des participants et des tickets.

## Fonctionnalités
- Gestion des événements
- Inscription des participants
- Gestion des conférenciers
- Réservation des salles
- Billetterie et paiement
